üè† Running AI Locally - Complete Guide

Ollama

Free

A tool for running large language models (like Llama 3 or DeepSeek) locally on your own machine. Perfect for privacy-conscious development and offline work.

üì• Installation

üçé macOS
brew install ollama
üêß Linux
curl -fsSL https://ollama.com/install.sh | sh
ü™ü Windows

Download from: ollama.com/download/windows

üöÄ Quick Start

1. Start Ollama
ollama serve
2. Pull a Model (e.g., Llama 3)
ollama pull llama3
3. Run the Model
ollama run llama3

üåü Popular Models

Llama 3
ollama pull llama3

Meta's latest, 8B params

DeepSeek Coder
ollama pull deepseek-coder

Specialized for coding

Mistral
ollama pull mistral

Fast & efficient, 7B params

CodeLlama
ollama pull codellama

Code-focused Llama variant

    "model": "llama3"
